动画3种：
position: relative; 把页面元素设置为可以移动，相对坐标
	a. 移动
	b. 拉伸
	c. 旋转

animation: mymove 2s infinite;

animation关键字，标识是一个动画
mymove 动画脚本名称
2s 动画耗时
infinite; 反复执行，循环效果，默认动画只执行一遍
@keyframes mymove{  关键帧
  from{left: 0px;}
  to{left: 300px;}
}
css3：用户定义一个开始帧，定义一个结束帧，它会自动把中间过程连接起来


transition: width 2s;
transition 拉伸关键字
width 拉宽，height 拉高
2s 执行动作的耗时

hover 鼠标移入触发事件
.transitionbox:hover{
  width: 300px;
}
把长度拉伸到300



animation: earth-rotation 5s linear 0s infinite;
animation:  关键字
earth-rotation 脚本名称
5s 执行耗时
linear 线型轨道
0s 启动时等待时间
infinite 重复播放

@keyframes earth-rotation{
  from{ transform: rotate(0deg);}
  to{ transform: rotate(360deg);}
}

transform变相，rotate 旋转
0deg 从0度开始 360deg ，转一圈，

.earth-rotation:hover{
  animation-play-state: paused;
}

hover鼠标移入，播放状态，暂停
