基本运算符：
  • 加法：{{ 3+3 }}
  • 拼接：{{ "3"+"3" }}、{{ 3+"3" }}、{{ '3'+"3" }}
  • 减法：{{ "3"-"3" }}、{{ 3-2 }}
  • 乘法：{{ "2"*"3" }}、{{ 3*2 }}
  • 除法：{{ 3/3 }}、{{ 3/"3" }}、{{ 3/"2" }}、{{ 10/3 }}
    三目运算：
  • {{ age>18 ? "老男人":"小鲜肉" }}
    字符串：
  • 信息内容：{{ message }}
  • 信息长度：{{ message.length }}
  • 截串：{{ message.substring(1,3) }}
    数组：
  • 数组：{{ hobbys}}
  • 长度：{{ hobbys.length}}
  • 元素：{{ hobbys[1] }}
    对象：
  • {{ girlFriend.name}}
  • {{ girlFriend.age}}
  • {{ girlFriend.msg}}
